# Bolo - Video Calling Application

## Setup instruction

### Clone repo

```bash
git clone https://github.com/
cd x
```

### Install dependencies

```base
npm install
```

### Setup firebase

1. Create firebase project in firebase console
2. Activate firestore
3. Add firebase config to .env file

```env
VITE_API_KEY=
VITE_AUTH_DOMAIN=
VITE_PROJECT_ID=
VITE_STORAGE_BUCKET=
VITE_MESSAGING_SENDER_ID=
VITE_APP_ID= a
```

### Start dev server

```bash
npm run dev
```
